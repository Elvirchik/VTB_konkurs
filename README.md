# ВТБ Финансовый трекер

Веб‑приложение на Django для учёта личных финансов, анализа расходов и постановки финансовых целей.

## Технологии

- Python 3.10+
- Django 5.x
- SQLite
- Bootstrap 5 (CDN)
- Chart.js (CDN)
- pandas
- python-dotenv

## Установка и запуск

- python -m venv .venv
- pip install django pandas
- python manage.py migrate
- python manage.py runserver
